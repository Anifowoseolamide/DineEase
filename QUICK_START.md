# DineEase Restaurant Website - Quick Start Guide

## 🚀 Get Started in 5 Minutes

### 1. Start the Website
```bash
./start.sh
```

### 2. Access Your Website
- **Main Website**: http://127.0.0.1:8000/
- **Admin Panel**: http://127.0.0.1:8000/admin/
- **Admin Login**: admin / admin123

## 🛠️ Quick Customization

### Change Restaurant Information
1. Go to Admin Panel → Core → Restaurant Information
2. Edit the restaurant details
3. Save changes

### Add Menu Items
1. Go to Admin Panel → Menu → Menu Items
2. Click "Add Menu Item"
3. Fill in details and upload images
4. Save

### Manage Reservations
1. Go to Admin Panel → Reservations → Reservations
2. View all booking requests
3. Update status (pending, confirmed, etc.)

## 📱 Features Overview

- **Home Page**: Hero section, about preview, popular dishes
- **Menu**: Organized by categories with dietary badges
- **Reservations**: Easy online booking with validation
- **Gallery**: Photo showcase with lightbox effects
- **About**: Restaurant story and information
- **Contact**: Multiple ways for customers to reach you

## 🔧 Technical Details

- **Framework**: Django 5.2.5
- **Database**: SQLite (easy to switch to PostgreSQL)
- **Frontend**: Bootstrap 5 + Custom CSS
- **Forms**: Django Crispy Forms
- **Images**: Pillow for processing
